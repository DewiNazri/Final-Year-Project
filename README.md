# Final-Year-Project
PERSONALIZED HAIR PRODUCTS RECOMMENDATION SYSTEM BASED ON LIFESTYLE FACTOR USING HYBRID FILTERING

## Features
- Personalized hair product recommendations
- Content-based filtering with **TF-IDF and Cosine Similarity**
- User-friendly interface for inputting hair profile
- Web-based system using **Python, Flask, HTML, CSS, JavaScript**

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Supabase 
- Data analysis: TF-IDF, Cosine Similarity
